# 入力設定

入力設定セクションでは、素材となる動画ファイルまたは PNG 連番フォルダと、再生 FPS を設定します。

## 入力モードの選択

- UI ラベル: 「入力モード」 → 「動画ファイル」 / 「PNG連番フォルダ」
- 動画ファイルからの入力には FFmpeg が必要です
- PNG 連番は、事前に自分で動画から書き出した PNG を使う場合に選びます

## 動画ファイル入力

### 動画ファイルの指定

- UI ラベル: 「動画ファイル」（ObjectField）
- Assets フォルダ内にある動画ファイルをドラッグ＆ドロップで指定します
- 対応形式: mp4, webm, mov, avi, mkv
- FFmpeg が見つからない場合は警告が表示されます
  → 参照: guide-source/ffmpeg-setup.md

<!-- IMG: 動画ファイルを指定した直後の入力設定セクション（元動画情報が表示されている） -->

### 書き出し FPS

- UI ラベル: 「書き出し FPS」
- 動画を PNG に変換する際の FPS を指定します（元動画の FPS とは別です）
- FPS を低くするとフレーム数が減り、テクスチャ容量が軽くなります
- デフォルトは 8fps です。多くの場合これで十分です
  → 参照: guide-source/fps-and-length-guide.md

### トリミング

- UI ラベル: 「トリミング」 → 「開始（秒）」 / 「長さ（秒）」
- 動画の一部だけを使いたい時にチェックを入れます
- 開始秒と使用する長さ（秒）を指定します
- 開始＋長さが動画の長さを超えている場合は警告が表示されます
- トリミング後の推定フレーム数が自動表示されます

## PNG 連番フォルダ入力

### 入力フォルダの指定

- UI ラベル: 「入力フォルダ」（ObjectField）
- PNG 連番が入った Assets 内のフォルダを指定します
- フォルダ内の PNG ファイルが自動的にフレームとして読み込まれます

<!-- IMG: PNG 連番フォルダを指定した状態の入力設定セクション -->

### PNG連番 FPS

- UI ラベル: 「PNG連番 FPS」
- PNG を書き出した時の FPS を入力します（元動画の FPS ではありません）
- 書き出し時の FPS が分からない場合は FPS 計算機で逆算できます
  → 参照: guide-source/fps-and-length-guide.md

### FPS 計算機

- UI ラベル: 「FPS 計算機」（Foldout、デフォルト閉じ）
- PNG の枚数と元動画の長さ（分・秒）から FPS を逆算できます
- 「PNG 枚数」: 手入力するか「入力フォルダから取得」ボタンで自動カウント
- 「動画 分」「動画 秒」: 元動画の長さを入力
- 「FPS に適用」ボタンで計算結果を「PNG連番 FPS」に反映できます
