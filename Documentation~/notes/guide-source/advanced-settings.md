# 上級設定

通常は変更する必要がない設定です。デフォルトのままで問題なく使えます。

## 上級設定の開き方

- UI ラベル: 「上級設定」（Foldout、デフォルト閉じ）
- ウィンドウ最下部にあります

<!-- IMG: 上級設定 Foldout を展開した状態（全項目が見える） -->

## 最大解像度（動画入力のみ）

- UI ラベル: 「最大解像度」 → 256 / 512 / 1024
- FFmpeg で動画を PNG に書き出す際の最大ピクセルサイズです
- デフォルト 512 で多くの場合十分です
- 元動画の解像度が高い場合でも、ここで指定したサイズに縮小されます
- 値を下げるとテクスチャ容量が軽くなりますが、画質は落ちます

## 最大シートサイズ

- UI ラベル: 「最大シートサイズ」 → 512 / 1024 / 2048 / 4096
- 1 枚のスプライトシートの最大ピクセルサイズです
- デフォルト 2048 で多くの場合十分です。**通常は変える必要はありません**
- シートサイズを変えても総テクスチャ容量はほぼ変わりません（1 枚が大きくなるか、枚数が増えるかの違い）
- Unity の Import Settings にある「Max Size」とは別の設定です
  → 転用元: technical/max-sheet-size-design.md の「シートサイズの影響は実は小さい」「最大シートサイズ vs Unity の Max Size」

## 自動分割（MultiPageSequence のみ）

- UI ラベル: 「自動分割」（Toggle）+ 「1ページ最大フレーム数」
- 自動分割 ON（デフォルト）: シートサイズから最適なフレーム数を自動計算します
- 自動算出値が HelpBox に表示されます
- 手動で調整することも可能ですが、通常は自動のままで問題ありません

## コンソールログ

- UI ラベル: 「コンソールにログを出力」
- デフォルト OFF です
- 生成過程の詳細を Unity の Console ウィンドウで確認したい場合に ON にします
- トラブルシューティング時に役立ちます
