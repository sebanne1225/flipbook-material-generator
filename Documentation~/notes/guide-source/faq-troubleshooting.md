# FAQ・トラブルシューティング

## 動画が止まって見える / カクつく

- FPS が高すぎてフレーム数が膨大になっている可能性があります
- 「書き出し FPS」を下げてみてください（5〜8fps 推奨）
- 「動画が長い ≠ ダメ」です。FPS を下げればフレーム数を抑えられます
  → 参照: guide-source/fps-and-length-guide.md

## テクスチャメモリーが大きすぎる

- フレーム数を減らすのが一番効果的です
  - 「書き出し FPS」を下げる
  - 「トリミング」で動画の一部だけ使う
- 上級設定の「最大解像度」を下げる（512 → 256）のも有効です
- 実行セクションに「推定テクスチャメモリー増加」が表示されるので、Generate 前に確認してください

## 「FFmpeg がパスに見つかりません」と表示される

- FFmpeg がインストールされていないか、PATH が通っていない状態です
- コマンドプロンプトで `ffmpeg -version` が動くか確認してください
  → 参照: guide-source/ffmpeg-setup.md

## 「フレーム数が収容上限を超えています」と表示される

- Texture2DArray / LilToon モードではフレーム数に上限があります
- 対処法:
  - 「トリミング」で使用区間を短くする
  - 「書き出し FPS」を下げる
  - 「出力モード」を MultiPageSequence に切り替える（フレーム上限なし）

## 「lilToon がプロジェクトに導入されていません」と表示される

- LilToon モードは lilToon シェーダーが必要です
- VCC から lilToon を導入するか、Texture2DArray モードを使ってください

## EX メニューに表示されない

- Modular Avatar（MA）がプロジェクトに導入されているか確認してください
- 「Prefab も生成する」が ON になっているか確認してください
- MPS モード: 「MA Merge Animator」と「MA Object Toggle」が ON になっているか確認してください
- 非 MPS モード: 「MA Object Toggle を追加」が ON になっているか確認してください
- 生成した Prefab がアバタールートの子オブジェクトに配置されているか確認してください

## 生成したものを削除したい

- 出力設定セクションの「生成済みスロット一覧」を開いて「削除」ボタンで一括削除できます
- ゴミ箱には送られず完全削除されます

## Quad が裏向きに見える

- Scene View のカメラ角度による見え方の問題です
- 実際の描画はすべてのモードで両面対応済みなので正常です
- Play モードの Game View で確認するのが確実です
