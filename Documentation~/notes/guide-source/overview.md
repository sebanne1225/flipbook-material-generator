# このツールでできること

## 何を作るツール？

- 動画ファイルや PNG 連番から「フリップブック」を自動生成するツールです
- フリップブック = パラパラ漫画のように画像を切り替えて再生するアニメーション
- スプライトシート・マテリアル・Prefab・EX メニュー連携まで一括で生成できます
- 元アバターには直接変更を加えない非破壊設計です

## 3 つの出力モード

### MultiPageSequence（おすすめ）
- 長い動画もフレーム制限なしで扱える
- 音声との同期に対応
- EX メニューから再生 ON/OFF・ループ・リセットを制御できる

### Texture2DArray
- 軽量で短いループアニメに最適
- フレーム数に上限あり（シートサイズに依存）

### LilToon
- lilToon シェーダーの内蔵機能で再生するため、シェーダー追加が不要
- lilToon が導入されている環境向け
- フレーム数に上限あり

→ 迷ったら MultiPageSequence を選べば大丈夫です。詳しくは → 参照: guide-source/output-modes.md

## こんな人向け

- アバターに動画や GIF 的なアニメーションを載せたい
- 推し画像のスライドショーを作りたい
- BGM 付きの映像演出をアバターに仕込みたい
- フリップブックの仕組みを知らなくても、動画を指定するだけで使いたい

## 安心ポイント

- **非破壊**: 元アバターのデータには一切変更を加えません
- **Dry Run**: 生成前に「何が作られるか」をダイアログで確認できます
- **MA は optional**: Modular Avatar がなくても Prefab 生成は可能です
- **容量確認**: 生成前に推定テクスチャメモリー増加量が表示されます
