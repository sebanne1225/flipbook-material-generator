# 出力設定

出力設定セクションでは、出力モードの選択と、生成ファイルの保存先を設定します。

## 出力モード

- UI ラベル: 「出力モード」 → Texture2DArray / LilToon / MultiPageSequence
- 選択中のモードに応じた説明が HelpBox に表示されます
- Texture2DArray / LilToon では「最大 N フレーム ≈ X.X 秒」の秒数ガイドも表示されます
- 各モードの詳しい違いは → 参照: guide-source/output-modes.md

## 出力先

- UI ラベル: 「出力先」 → 「元ソース直下」 / 「ツール共通フォルダ」 / 「フォルダを指定」
- **元ソース直下**: 入力フォルダの直下に `Generated_Flipbook/` を作成します
- **ツール共通フォルダ**（デフォルト）: `Assets/Sebanne/FlipbookMaterialGenerator/Generated_Flipbook/` に生成します
- **フォルダを指定**: 任意のフォルダを選択できます（「出力フォルダ」フィールドが表示されます）
- 「現在の出力先」に実際の保存パスが表示されるので、生成前に確認できます

## 出力名とスロット

- UI ラベル: 「出力名」
- 動画やフォルダを指定すると自動でファイル名が入力されます。手動で変更も可能です
- UI ラベル: 「スロット」 → 「自動（新規作成）」 / 既存スロット名
- スロット方式: `Generated_Flipbook/01_名前/` のように番号付きフォルダで管理されます
- 「自動（新規作成）」を選ぶと次の番号で新しいフォルダが作られます
- 既存スロットを選ぶと、そのフォルダの中身が上書きされます（警告が表示されます）

## スロットブラウザ

- UI ラベル: 「生成済みスロット一覧」（Foldout）
- 過去に生成したスロットの一覧を確認できます
- 各スロットに表示される情報:
  - シート数・マテリアル数・Prefab の有無・Audio の有無
- 操作:
  - スロット名をクリック → スロットドロップダウンで選択状態にする
  - 「フォルダを開く」 → Project ウィンドウでフォルダをハイライト
  - 「削除」 → 確認ダイアログの後、スロット内のファイルを完全削除（ゴミ箱に送られません）
