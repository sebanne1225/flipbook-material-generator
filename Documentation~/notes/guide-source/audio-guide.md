# 音声つきフリップブック

この章では、フリップブックに音声を追加する方法を説明します。映像と音声を同期させたい場合のモード選びについても解説します。

## 音声を追加するには

Prefab / MA 設定セクションで **音源を追加** を ON にして、**AudioClip** のフィールドに WAV ファイル等をドラッグ＆ドロップで指定します。

動画ファイルを入力に使っている場合は、**動画から音声を抽出** ボタンを使うと、動画から音声を自動で取り出して設定できます。自分で WAV を用意する手間が省けます。

## 音声同期の違い

音声と映像の同期ができるかどうかは、出力モードによって異なります。

| モード | 音声同期 | 音声の使い方 |
|---|---|---|
| MultiPageSequence | **対応** | 映像と音声がズレずに再生されます |
| Texture2DArray | 不可 | BGM・環境音など同期不要の用途に使えます |
| LilToon | 不可 | BGM・環境音など同期不要の用途に使えます |

MultiPageSequence は AnimatorController（Unity のアニメーション制御の仕組み）で映像を再生しているため、音声との同期が可能です。映像と音を合わせたい場合はこのモードを選んでください。

Texture2DArray と LilToon はシェーダー内部の時間で再生するため、AnimatorController とは連動できません。音声を追加すること自体は可能ですが、映像との同期は保証されません。ループ BGM や環境音など、タイミングを合わせる必要がない音声を添えたい場合に使えます。

## 音声抽出機能

**動画から音声を抽出** ボタンは、入力モードが「動画ファイル」の場合に表示されます。

このボタンをクリックすると、FFmpeg を使って動画ファイルから音声が WAV 形式で抽出されます。抽出された WAV はスロット内の `Audio/` フォルダに保存され、**AudioClip** に自動設定されます。

トリミング設定がある場合は、音声にも同じトリミングが適用されます（「トリミング設定が音声にも適用されます」というメッセージが表示されます）。

音声トラックがない動画の場合は「この動画に音声トラックはありません。」という警告が表示されます。

## 音声追加時の注意

**音源を追加** が ON の状態で **AudioClip** が未指定のまま Generate すると、音声なしで生成されます。実行セクションに「AudioClip が指定されていません」という警告が表示されるので、意図的に音声なしにしたい場合以外は AudioClip を設定してください。
