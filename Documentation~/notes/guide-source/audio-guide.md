# 音声つきフリップブック

フリップブックに音声を追加して、映像と一緒に再生できます。

## 音声を追加するには

- Prefab / MA 設定セクションで「音源を追加」を ON にします
- 「AudioClip」フィールドに WAV ファイル等をドラッグ＆ドロップで指定します
- 動画入力の場合は「動画から音声を抽出」ボタンで動画から自動抽出もできます

## 音声同期の違い

音声と映像の同期ができるかどうかはモードによって異なります。

| モード | 音声同期 | 音声の使い方 |
|---|---|---|
| MultiPageSequence | **対応** | 映像と音声がズレずに再生される |
| Texture2DArray | 不可 | BGM・環境音など同期不要の用途に |
| LilToon | 不可 | BGM・環境音など同期不要の用途に |

- MultiPageSequence は Animator で映像を制御しているため、音声との同期が可能です
- Texture2DArray / LilToon はシェーダーの内部時間で再生するため、Animator と連動できません
- 音声同期が必要な場合は MultiPageSequence を選んでください
  → 転用元: technical/playback-control-integration.md の「3モード vs MultiPageSequence の Audio 分岐」

## 音声抽出機能

- UI ラベル: 「動画から音声を抽出」ボタン
- 動画入力モード（「入力モード」が「動画ファイル」）の場合に表示されます
- FFmpeg を使って動画から音声を WAV 形式で抽出します
- トリミング設定がある場合、音声にも同じトリミングが適用されます
- 抽出された WAV はスロット内の `Audio/` フォルダに保存されます
- 音声トラックがない動画の場合は警告が表示されます

## 音声追加時の注意

- 「音源を追加」が ON で AudioClip が未指定の場合、実行セクションに警告が表示されます
- その状態でも Generate は可能ですが、音声なしで生成されます
