Flipbook Material Generator
BOOTH 配布パッケージ案内
version: 1.0.0

このツールは、動画ファイルまたは PNG 連番フォルダから、
VRChat 向けのフリップブックアニメーション素材を自動生成する Unity Editor ツールです。

主な方針:
- まず Dry Run で生成内容を確認します
- このツールは非破壊で動作し、元アバターには直接変更を加えません
- 必要な素材だけを新規 asset として生成します

最初にやること:
1. repo の README を確認してください
2. まずは VCC / VPM で導入してください
3. 導入後は Dry Run を試してください

導入の基本:
- 主導線は VCC / VPM です
- Git URL 導入は補助扱いです

GitHub repo:
https://github.com/sebanne1225/flipbook-material-generator

VCC に追加する URL:

参考ページ (VCC 追加先ではありません):
